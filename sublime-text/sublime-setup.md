Setup Sublime Text 2
====================

Installing Sublime Text 2
-------------------------

1. Add webupd8team apt repository for Sublime Text 2

        (sudo) add-apt-repository ppa:webupd8team/sublime-text-2
        (sudo) apt-get update
        (sudo) apt-get install sublime-text
