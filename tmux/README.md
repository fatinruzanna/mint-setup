Setup Tmux
==========

To install Tmux, use the following command:

		(sudo) apt-get install tmux


Configure Tmux
---------------

1. Create .tmux.conf file

		touch ~/.tmux.conf


2. Add to your .tmux.conf file with the contents in _tmux.conf_ file.
